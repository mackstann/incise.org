Service name for all methods is org.mpris.corn. Example calls using dbus-send: 
$ dbus-send --session --type=method_call --print-reply --dest=org.mpris.corn / org.freedesktop.MediaPlayer.Identity
method return sender=:1.684 -> dest=:1.785 reply_serial=2
   string "corn 0.0.0"
$ dbus-send --session --type=method_call --dest=org.mpris.corn /Player org.freedesktop.MediaPlayer.Next
  • Interface: org.freedesktop.MediaPlayer (aka MPRIS)
    • Path: /
      • Method: Identity() -> (program/version string)
      • Method: Quit()
      • Method: MprisVersion() -> (struct { uint16 major, uint16 minor }) XXX broken
    • Path: /Player
      • Method: Next()
      • Method: Prev()
      • Method: Pause()
      • Method: Stop()
      • Method: Play()
      • Method: GetCaps() -> (bitmask of capabilities)
      • Method: VolumeSet(int 0-100)
      • Method: VolumeGet() -> (int 0-100)
      • Method: PositionSet(int 0-100)
      • Method: PositionGet() -> (int 0-100)
      • Method: Repeat(boolean to repeat current track or not)
      • Method: GetMetadata() -> (hash/dict of current song's metadata)
    • Path: /TrackList
      • Method: DelTrack(int track)
      • Method: AddTrack(string uri, boolean playnow) -> (int failed)
      • Method: GetLength() -> (int number of tracks)
      • Method: GetCurrentTrack() -> (int track)
      • Method: SetLoop(boolean) # loop at end of playlist or not
      • Method: SetRandom(boolean) # play randomly or not
      • Method: GetMetadata(int track) -> (hash/dict of metadata for $track)
  • Interface: org.corn.CornPlayer (things that should be in MPRIS but aren't.. yet)
    • Path: /Corn
      • Method: Clear() # clear playlist
      • Method: PlayTrack(int track) # play given track
      • Method: Move(int from, int to) # move track at $from to $to
