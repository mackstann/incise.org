py-libmpdclient2 is a Python library for making MPD (Music Player Daemon) clients.

This library is not maintained and has been superceded by the python-mpd library. New projects should use python-mpd.

Download/Install

  • Debian and Ubuntu: apt-get install python-mpdclient
  • Via the PYPI: easy_install py-libmpdclient2 or pip install py-libmpdclient2
  • Tarball: http://incise.org/files/dev/py-libmpdclient2-1.0.tgz
